HOW TO USE
------------

1.) replace either 'screenshot.png' or 'text.txt' with your information. examples of what you should put in the files are in the 'examples' folder.

2.) run 'scheduler.exe', enter a name for your schedule and then input either 'TEXT' or 'SCREEN' based on the prompt

3.) take the 'schedule.csmo' file that is outputted to this same directory to the website 'https://freecollegeschedulemaker.com/'

4.) press 'import schedule' and drag the 'schedule.csmo' file into the designated box

5.) fix any minor errors and press 'save as image' to keep your new formatted schedule!

THINGS THAT CAN GO WRONG
----------

1.) i'm using a python wrapper for google's tesseract ocr. what this means is that the image recognizing software is really finnicky so if that doesn't work the text function will probably work a lot better.

2.) if your class has TBA for its information, then in the schedule its being put in as meeting between 12:00PM and 12:30PM on no days. so its not displaying on the schedule. this is intentional because you don't have a time lol. you can edit and fix it by pressing 'edit course' on the website and navigating to the course that you need to fix.
